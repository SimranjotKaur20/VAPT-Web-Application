# Vulnerability Assessment and Penetration Testing (VAPT) on a Web Application

## Project Overview

This project demonstrates the Vulnerability Assessment and Penetration Testing (VAPT) process for a web application using industry-standard cybersecurity tools. The objective is to identify, analyze, and document common web application vulnerabilities and recommend appropriate remediation techniques.

## Objectives

- Perform information gathering and reconnaissance
- Enumerate web application resources
- Identify vulnerabilities using automated and manual techniques
- Assess risk using CVSS
- Recommend mitigation strategies

## Methodology

1. Planning and Authorization
2. Information Gathering
3. Website Enumeration
4. Fingerprinting
5. Vulnerability Assessment
6. Manual Penetration Testing
7. Exploitation
8. Risk Assessment
9. Remediation

## Tools Used

- Nmap
- WhatWeb
- Gobuster
- Burp Suite
- Nikto
- SQLMap
- Nuclei
- Wireshark

## Technologies

- Kali Linux
- Linux Command Line
- HTTP/HTTPS
- CVSS
- OWASP Methodology
